# 下一步部署指引

1. 解壓縮專案。
2. 上傳全部檔案到 GitHub repository：`fc781117-jason/tw-stock-ai-monitor`。
3. 到 Vercel 匯入 GitHub repository。
4. 在 Vercel 專案設定環境變數。
5. Deploy。
6. 取得 Vercel 網址後，回到 Supabase / Google Cloud 補正式 Redirect URL。

## 注意

不要把以下資訊上傳 GitHub：

- OpenAI API key
- Supabase secret key
- Google OAuth client secret
- Database password
